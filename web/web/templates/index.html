Welcome to DTANM!

As a contestant, you are responsible for fixing your program, and developing "attacks", or test vectors that your program handles correctly, but that other teams' programs may not.

  1. Read up on your challenge in the challenge docs
  2. Download the challenge according to the docs on the submit program page
  3. Fix the bugs in your program according to the docs from step 1
  4. Upload the fixed program according to the docs on the submit program page
  5. Create attacks against other teams' programs on the create attack page
  6. Track your team's score on the scoreboard!

Good luck!
